## 📁 Repository Blueprint

```text
CryptoPro-Core/
├── .vscode/               # Environment Configuration
├── crypto-website/        # 🌐 Modern Web Interface
│   ├── index.html         # Dashboard Entry
│   ├── script.js          # Interactive UI Logic
│   └── style.css          # Glassmorphism Stylesheet
├── CryptoGUI.java         # 🖥️ Desktop UI Controller
├── CryptoProject.java     # ⚙️ Core Logic Engine
├── .gitignore             # Optimized Exclusions
└── LICENSE                # MIT Framework
```

# 🚀 Installation & Setup
## ☕ Desktop Core
```text
Bash

# Clone the repository:
git clone [https://github.com/AND-SHAL-0813/CryptoPro-Core.git](https://github.com/AND-SHAL-0813/CryptoPro-Core.git)

# Compile and Run:
javac CryptoProject.java CryptoGUI.java
java CryptoGUI
```
---
## 🌍 Web Interface
Navigate to the crypto-website directory and open index.html in any modern browser. Currently optimized for Chrome and Edge.
